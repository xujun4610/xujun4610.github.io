使用前说明

请到Android系统 设置> 应用> “固定资产”App> 权限，开启页面中所有访问权限。

登录

属于用户名和密码，点击【登录】，进入App主屏幕。

> 注意：如果第一次使用，请先点击下方的【软件设置】

软件设置

设置资产API地址、扫描枪参数等设置

资产API地址

输入格式：例如,http://192.168.1.123:10009

> 注意：最后不带斜杠

扫描头类型

设置扫描头类型。目前仅支持两种：

  • 2D扫描头（需扫描枪带有设置App）

  • Zebra® RFID（EMDK套件+TC系列设备）

扫描键设置

当使用“2D扫描头”时候，此处设置会生效。

> 长触输入框，进入扫描键设置窗口。

在线/离线设置

此功能作废。

主功能页面

登录之后，显示软件主功能页面。此处列出“固定资产”主功能列表。

常用功能概览：

功能名称 描述
数据下载 从资产服务器上下载数据到本地
资产列表 查看、搜索、定位资产卡片
资产明细 查看资产卡片信息以及上传图片类型附件
盘点计划 盘点计划列表页面，后续进入可盘点
盘点上传 如果存在计划未能成功上传，这里重新上传
使用说明 此软件的使用手册
关于软件 关于此软件的版本信息、数据库版本等信息
软件设置 配置API地址、扫描头、扫描键等内容
图片上传 上传图片附件（该功能不直接在主屏幕显示）

数据下载

该功能为离线数据下载。目前可下载如下内容：

  • 资产卡片信息

  • 用户信息

  • 部门信息

    Tips: 联络网络后，点击按钮即可下载。

资产列表

下载到的资产卡片数据。（离线模式）
该界面可以搜索，定位，资产卡片。

Tips: 由于数据量太大，因此采用分页方式，请使用底部的分页组件。

点击其中一条“资产卡片”信息。进入《资产卡片》明细页面。

资产明细

查看卡片详细登记的信息。该页面有如下功能：

  • 查看卡片信息

  • 未上传的图片附件，进行拍照上传

  • 查看已上传的图片附件

  • 新增/修改资产卡片*

注意：


* 这里的新增、修改资产卡片根据用户的需要，进行删减，程序将来未来版本中加入启用和关闭的设置。

图片上传

如果需要拍摄资产的照片加以区分，这里提提供了入口。

注意：

  1. 由于机能有限，最大上传5张图片。

  2. 每张图片最大5Mb（1Mb = 1024Kb）。

  3. APP临时拍摄的图片长期保存在【内部存储】\DCIM\Pictures 文件夹下。

  4. 上传的时候请保持网络通畅，且服务器同时没有非常大的负载压力。

盘点计划

该功能查看您所有的盘点计划，该页面有如下功能：

  • 下载数据：从服务器上下载最新的计划数据。

  • 搜索：如果页面数据量太大，可以搜索，快速定位。

选择一行“未完成”状态盘点计划主记录数据，进入《盘点计划》明细页面。

> Tips: 从列表下拉可以刷新数据！

盘点计划明细（从“盘点计划”进入）

该页面含有如下菜单功能：

  • 盘点：未完成计划，点击右上角菜单> “盘点”

  • 获取更新：获取服务器上最新版本的该计划明细数据。

    提示：盘点页面的加载根据本app设置的“扫描头类型”决定。

盘点扫描界面（2D扫描头）

进入界面后，点击输入框，即可开始扫码工作。

> Tips：1. 扫描前请在本App设置好对应的扫描键。2. 扫描的条码最后必须带回车符（KeyCode = 13）。
盘点操作步骤

  1. 点击输入框，开始扫码。

  2. 您觉得扫描完成后，点击右上角的上传按钮，开始执行上传。

    Tips: 如果没有网络，则本次扫描的结果暂存至本地，待有网络连接后到《盘点上传》功能进入，重新上传。

盘点扫描界面（RFID发射器，需设备支持）

盘点操作步骤

  1. 连接上RFD2000套件后，点击界面上Switch开关按钮。

  2. 等待设备激活后，按动RFD2000的扳机，开始扫描。

  3. 您觉得扫描完成后，点击右上角的上传按钮，开始执行上传。

    Tips: 如果没有网络，则本次扫描的结果暂存至本地，待有网络连接后到《盘点上传》功能进入，重新上传。

盘点上传

该功能查看未能成功上传，且已有本地扫描记录的盘点计划列表。

选择一行“未完成”状态盘点计划主记录数据，进入《盘点计划》明细页面。

> Tips: 从列表下拉可以刷新数据！

盘点计划明细页面（从“盘点上传”进入）

该页面含有如下菜单功能：

  • 获取更新：获取服务器上最新版本的该计划明细数据。
    若您的数据版本过旧，则需要点击此处

  • 重新上传：本地已扫描但未能成功上传的数据，进行重新上传。
    这里不进入盘点扫描界面，直接上传。
